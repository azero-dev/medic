# Gestor de Medicamentos

PWA para gestionar la medicación diaria. Permite crear medicamentos con sus horarios, visualizar un calendario semanal y recibir notificaciones cuando es hora de tomar cada medicamento.

## Funcionalidades

- **Calendario semanal** - Vista rápida de los medicamentos de cada día.
- **Vista diaria** - Lista detallada con botón para marcar como consumido.
- **CRUD medicamentos** - Crear, editar y eliminar medicamentos.
- **Notificaciones** - Recordatorios a la hora programada.
- **PWA** - Instalable en móvil, funciona offline.

## Tech Stack

- React + Vite
- IndexedDB para persistencia
- Service Worker para PWA
- Cloudflare Pages para deployment

## Cómo instalar la PWA desde el navegador

### Firefox

1. Abre la aplicación en Firefox
2. Toca el menú (⋮) en la esquina inferior derecha
3. Selecciona "Añadir a pantalla de inicio"
4. Confirma la instalación

### Chrome

1. Abre la aplicación en Chrome
2. Toca el menú (⋮) en la esquina superior derecha
3. Selecciona "Instalar aplicación" o "Añadir a pantalla de inicio"
4. Confirma la instalación

La PWA se instalará como una aplicación independiente, sin la barra de direcciones del navegador.
